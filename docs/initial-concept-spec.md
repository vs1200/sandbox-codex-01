# tetris-ren-ren 初期コンセプト

## ツール概要
- Tetris の連鎖を練習するツールです
- 4列RENと呼ばれる行消しを連続で行う配置を練習できます

## 参考実装
https://shiwehi.com/tetris/game/i4lr.php
https://shiwehi.com/tetris/game/i4lr.js?date=20241203

## 技術スタック
- vite, React, zustand, tailwind v4, biome

## 設計指針
- 参考実装は1ファイルだが古い時代の実装なので、テトリス関連のロジックは参考になるが、それ以外は参考にしない
  - 多くの範囲を modernize する意識で作業する
  - SOLID原則を重視してコンポーネント分割も丁寧に行う
    - 将来的には、段階的に機能を追加していく必要があるため
- css 関連も modarnize を意識する
  - `@themes` ディレクティブを利用する
  - (必要があれば) nested css を利用する

## 要件
- 参考実装で実現されている機能は一通り実装する
- 固定サイズ画面ではなく、ウィンドウの大きさに合わせて最適な表示になるようにする
- デザインはダークモードよりで落ち着いた感じが良いが、ミノの色はわかるようにする
